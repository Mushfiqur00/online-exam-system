Examora - Online Exam System 🎓

Examora is a secure, robust, and automated web-based digital examination platform built with Python (Flask). It streamlines the academic assessment process by offering automated scheduling, real-time timers, instant auto-evaluation for objective questions, and an analytical dashboard for educators.

 🚀 Key Features
* Role-Based Access Control (RBAC):** Secure and isolated dashboards for Admins (Teachers) and Students.
* Smart Exam Management:** Create exams, manage dynamic question banks, and assign them to specific student groups.
* Strict Exam Environment:** Server-synced real-time countdown timer, auto-submission at `00:00`, and basic anti-cheat mechanics (copy-paste block, tab-switch detection).
* Instant Evaluation:** Automated grading algorithm for Multiple Choice Questions (MCQs).
* Analytics Dashboard:** Visual performance insights and submission tracking using Chart.js.

 🛠️ Technology Stack
* Backend: Python 3, Flask, Werkzeug Security
* Database: SQLite / PostgreSQL, SQLAlchemy (ORM)
* Frontend: HTML5, CSS3, JavaScript, Bootstrap 5
* Data Visualization: Chart.js
